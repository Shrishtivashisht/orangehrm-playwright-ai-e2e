
# OrangeHRM – Prompt‑Driven AI Automation Framework (Playwright + TypeScript)
Status: Stable | Production-style E2E | Real-world SPA handling
This project demonstrates **AI‑assisted automation development** using a
**prompt‑driven workflow** with GitHub Copilot / LLMs.

## What makes this project different?
- Test requirements are written in **Markdown prompts**
- AI is used to **generate Page Objects and tests**
- Engineer reviews, refines, and owns final code
- Clean POM, stable locators, CI/CD ready

## Automated Workflow
Login → Dashboard → PIM → Add Employee → Verify Employee → Logout

## Tech Stack
- Playwright
- TypeScript
- Page Object Model (POM)
- GitHub Actions (CI)
- Prompt‑driven AI development

## How to Run
npm install  
npx playwright install  
npm test
